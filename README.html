びわ湖電池プロジェクトについて

📘 プロジェクト概要

「びわ湖電池」は、琵琶湖の水位を3cmだけ上げ下げすることで、 最大16GWhのエネルギーを蓄えることが可能な日本型の自然揚水蓄電システムです。 「自然揚水蓄電システム」とは、自然の地形を利用したダムに、 昼間の太陽光発電の電力で水をくみ上げ、夜間にそのダムから水力発電を行う仕組みです。 また、びわ湖が**淡水湖であること**は、揚水設備の腐食やスケール（堆積物）を抑えるうえでも大きな利点です。 これは**海水を用いる場合に比べ、設備寿命を延ばし、保守コストを大幅に下げられる可能性**を意味します。

  • 琵琶湖は日本最大の淡水湖で面積(669.26㎢)と貯水量(27.5㎦)があります
  • その水位3cmは水量に換算して約２０００万トン
  • その水量を300m持ち上げると16GWhのエネルギーを蓄えることができます
  • １GWは標準的な原子力発電機一機分で16GWhはその発電機で16時間運転する電力量です
  • びわ湖の周囲には最高1000メートルを超える山もあり揚水発電の好条件がととのっています

🌍 目的

  • ✅ 再生可能エネルギーの有効活用
    （太陽光、風力などの自然発電の補完の役割）
  • ✅ 夜間の電力安定供給
    （太陽光発電が得られない夜間に静かに水力発電する）
  • ✅ 災害時のブラックスタート電源
    （電力管内にまったく電源がないときにも運転開始できる「マッチ」の役割）
  • ✅ CO₂排出ゼロの未来社会づくり
    （石炭やLNGによる火力発電を減らし、再エネの真の自立を支える）
