びわ湖電池プロジェクトについて

📘 プロジェクト概要

「びわ湖電池」は、琵琶湖の水位を3cmだけ上げ下げすることで、 最大16GWhのエネルギーを蓄えることができる、日本型の自然揚水蓄電システムです。

  • 琵琶湖は日本最大の面積(669.26㎢)と貯水量(27.5㎦)があります
  • その水位3cmは水量に換算して約２０００万トン
  • その水量を300m持ち上げると16GWhのエネルギーを蓄えることができます
  • １GWは標準的な原子力発電機一機分で16GWhはその発電機で16時間運転する電力量です
  • びわ湖の周囲には最高1000メートルを超える山もあり揚水発電の好条件がととのっています

🌍 目的

  • 再生可能エネルギーの有効活用
  • 夜間の電力安定供給
  • 災害時のブラックスタート電源
  • CO₂排出ゼロの未来社会づくり

🧭 コンテスト

上池を自由に設計・提案していただく「びわ湖電池 上池設計コンテスト」を開催予定です。 市民・学生・技術者・研究者・AIすべて歓迎。
