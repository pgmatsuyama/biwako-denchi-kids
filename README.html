びわ湖電池プロジェクトについて

📘 プロジェクト概要

「びわ湖電池」は、琵琶湖の水位を3cmだけ上げ下げすることで、 最大16GWhのエネルギーを蓄えることが可能な日本型の自然揚水蓄電システムです。 「自然揚水蓄電システム」とは、自然の地形を利用したダムに、 昼間の太陽光発電の電力で水をくみ上げ、夜間にそのダムから水力発電を行う仕組みです。 また、びわ湖が**淡水湖であること**は、揚水設備の腐食やスケール（堆積物）を抑えるうえでも大きな利点です。 これは**海水を用いる場合に比べ、設備寿命を延ばし、保守コストを大幅に下げられる可能性**を意味します。

  • 琵琶湖は日本最大の淡水湖で面積(669.26㎢)と貯水量(27.5㎦)があります
  • その水位3cmは水量に換算して約２０００万トン
  • その水量を300m持ち上げると16GWhのエネルギーを蓄えることができます
  • １GWは標準的な原子力発電機一機分で16GWhはその発電機で16時間運転する電力量です
  • びわ湖の周囲には最高1000メートルを超える山もあり揚水発電の好条件がととのっています

🌍 目的

  • ✅ 再生可能エネルギーの有効活用
    （太陽光、風力などの自然発電の補完の役割）
  • ✅ 夜間の電力安定供給
    （太陽光発電が得られない夜間に静かに水力発電する）
  • ✅ 災害時のブラックスタート電源
    （電力管内にまったく電源がないときにも運転開始できる「マッチ」の役割）
  • ✅ CO₂排出ゼロの未来社会づくり
    （石炭やLNGによる火力発電を減らし、再エネの真の自立を支える）

🗾 滋賀県民への提言

滋賀県には現在、大規模な発電所は存在せず、主な電源は太陽光や小水力といった自然エネルギーに限られています。
自県内の電力自給力は、近隣府県と比べても限定的であり、他地域からの供給に依存しています。
しかし今、電力は「エネルギーのかなめ」として、AI・データセンター・脱炭素社会の推進に不可欠です。

電力の最大の特徴は「オンデマンド」。つまり、使うその瞬間に供給しなければならず、ためておくことが難しい点にあります。
災害やピーク需要、再生可能エネルギーの不安定性などを踏まえると、滋賀県にも持続可能で安定した地域内エネルギー基盤が求められます。

私たちの提案する「びわ湖電池」は、琵琶湖の水位を3cm変えるだけで、1,600万世帯分の夜間照明を支えるエネルギーを確保できる、地形と自然を活かした蓄電モデルです。

滋賀県から、未来を照らすエネルギー革新を。

🏢 データセンター誘致への展望

「びわ湖電池」によって滋賀県内の電力インフラが補強されれば、日本でも数少ない“地産地蓄”の電力供給地が実現します。
この特性を活かし、滋賀県は次世代データセンターの理想的な立地候補として、以下のような強みを発揮できます。

  • 安定した電源：びわ湖電池による揚水蓄電により、再エネ由来の夜間電力も安定供給可能
  • 冷却資源が豊富：琵琶湖の淡水は塩害がなく、蒸発冷却や水冷に理想的（北海道は空気冷却中心）
  • 交通・通信の要衝：関西・中京・北陸の中間で新幹線・高速道路・幹線通信が集中
  • 災害リスクが比較的低い：地震・津波・豪雪の影響が少なく、地形も安定
  • 土地利用の柔軟性：市街地から離れた山間部などに用地確保の可能性

これらの条件を総合すると、滋賀県は電源・冷却・地の利を兼ね備えた、日本でも数少ない「情報基盤インフラの中核候補地」です。
実際に、北海道では“雪による冷却”を武器にデータセンター誘致が進められていますが、滋賀県は通年使用可能な淡水冷却＋電力自立モデルという別軸の強みを提示できます。

将来、びわ湖周辺に「自然冷却型・ゼロカーボン・分散型」データセンター群が展開すれば、
滋賀県は電力と情報の両面で、近畿の“インフラ・ハブ”として新しい役割を担うことになります。

熱くなりがちなAIも、冷たい淡水にひかれて、静かなびわ湖のほとりへ引っ越してくるかもしれません。
